Typography in CSS

All fonts that can be brought on to a web page fall into one of the five generic font families:

  1. Serif — Serifs have a small stroke at the edges of each letter. They create a sense of formality and elegance.
  2. Sans Serif — Sans serifs have clean lines, no small strokes (aka serifs) attached. They create a modern and minimalistic look.
  3. Monospace — Monospace fonts have all letters with the same fixed width. They tend to create a mechanical look.
  4. Cursive / Script — These imitate human handwriting and calligraphy. They are good for decorative elements and headlines.
  5. Fantasy / Decorative — These are playful and decorative fonts. They are also good for headlines and decorative elements.

Locally hosted fonts need to be declared early on with the @font-face rule in order to be utilized.
