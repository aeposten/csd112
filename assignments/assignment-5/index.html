In modern web design, HTML tables should only be used to display tabular data such as schedules, statistics, or comparisons. Tables should not be used for page layout, as this was a common practice in early web development that has since been replaced by CSS layout techniques like Flexbox and Grid (I DEFINITELY used tables for layout purposes in my Geocities webpages back in the day). Using tables for layout reduces accessibility for screen reader users and makes code significantly more difficult to maintain.( MDN, "HTML Table Basics")
